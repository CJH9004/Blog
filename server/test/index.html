articles

  • get /articles/:id 获取单个文章
  • post /articles 创建文章
  • delete /articles/:id 删除文章
  • patch /articles/:id 修改文章
  • get /allArticles 获取所有文章
  • get /articles 获取所以已发布的文章（前台使用）
  • patch /pubArticles/:id 发布或不发布文章

tags

  • post /tags 创建标签
  • get /tags 获取所有标签
  • patch /tags/:id 修改单个标签
  • delete /tags/:id 删除单个标签
